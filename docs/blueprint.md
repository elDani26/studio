# **App Name**: GestionaTuDinero

## Core Features:

- Google Authentication: Enable users to securely log in using their Google account credentials.
- Transaction Management: Add new transactions with details like type (income/expense), category, optional description, transaction date and optional source account.
- Transaction Display: Show an intuitive UI for users to see all their transactions.
- Advanced Filtering: Implement advanced filters, by date range, category, transaction type, and origin account.
- Automated Summaries: Generate automated summaries that display stats such as percentage of income allocated to each category.
- Expense and Income Visualization: Visually display top expenses and income sources in a dashboard.
- Transaction Storage: Store transaction information in a database for future analysis and querying.

## Style Guidelines:

- Primary color: Soft Green (#8FBC8F), to evoke trust and growth
- Background color: Light Gray (#F0F0F0) for a clean and neutral backdrop.
- Accent color: Pale Blue (#ADD8E6) for interactive elements.
- Body and headline font: 'PT Sans', sans-serif, for a balance of modern style and warmth
- Use financial and category-related icons from a minimalist set (e.g. payment cards, cash, groceries).
- Clean and organized layout with a clear visual hierarchy for easy navigation.
- Subtle transitions and animations for user interactions (e.g., adding a transaction, filtering data).
